## ✦ About

**Swiftly** is a modern e-commerce web application designed to provide a **fast, clean, and user-friendly shopping experience**.

Built using **TypeScript, HTML, and Tailwind CSS**, this project focuses on delivering a **responsive interface** with strong attention to **UI design and usability**.

This project is developed as a **final semester assignment**, while also serving as a foundation for future development.

> *"Simple, fast, and intuitive — shopping made effortlessly."*

---

## ✦ Features

- 🛍️ Product listing and browsing  
- 🔍 Search and filtering system  
- 🛒 Shopping cart functionality *(planned / optional, sesuaikan)*  
- 📱 Fully responsive design  
- 🎨 Clean and modern UI  
- ⚡ Fast performance with Vite
